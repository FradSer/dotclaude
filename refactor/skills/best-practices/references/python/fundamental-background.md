## Background

**Impact: CRITICAL (Establishes the foundation for code style standards)**

Python is the main dynamic language used. This style guide is a list of *dos and don'ts* for Python programs, based on the Google Python Style Guide.

- **Formatters**: Use `Black` or `Pyink` to avoid arguing over formatting.
- **Goal**: Readability and maintainability.

Reference: [Google Python Style Guide](https://google.github.io/styleguide/pyguide.html)

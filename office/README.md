# Office Plugin

Patent application generation and intellectual property workflows with structured automation.

## Overview

The Office Plugin provides specialized tools for patent attorneys and IP professionals to generate patent applications, manage IP workflows, and handle patent-related documentation.

## Commands

### `/office:patent-draft`

Generates structured patent application drafts with claims and specifications.

**Usage:**
```bash
/office:patent-draft "Mobile Payment Authentication System"
```

**Features:**
- Patent claim structure generation
- Technical specification drafting
- Prior art consideration integration
- Patent drawing requirements support
- USPTO/USPTO format compliance

### `/office:patent-search`

Performs comprehensive prior art searches.

**Usage:**
```bash
/office:patent-search "Blockchain consensus algorithms"
```

**Features:**
- Patent database search integration
- Scientific literature review
- Prior art analysis
- Novelty assessment support

### `/office:ip-workflow`

Manages intellectual property workflow and documentation.

**Usage:**
```bash
/office:ip-workflow
```

**Features:**
- Patent application tracking
- Filing deadline management
- Document generation automation
- Workflow status monitoring

## Best Practices

- Provide detailed technical descriptions for better patent drafts
- Include specific implementation details and embodiments
- Use technical terminology appropriate for the patent field
- Review generated content for accuracy and completeness

## Requirements

- Access to patent databases (if using search features)
- Technical documentation for inventions
- Patent classification knowledge
